# Astro-DN

Astronomical image denoising model.

## Get started

1. Clone the repo:
   ```shell
   git clone https://github.com/oalvaroc/astro-dn.git
   cd astro-dn
   ```

2. Create Python virtual env:
   ```shell
   python -m venv .venv
   source .venv/bin/activate
   ```

3. Install `astro-dn` in the virtual env:
   ```shell
   pip install .
   ```
